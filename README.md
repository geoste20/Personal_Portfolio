## Part 2: Design

1. **Overall Style:**
   The design style is professional, modern, and minimal.

2. **Color Scheme:**
   I used a dark background with a blue accent color. The dark theme gives the website a clean and modern look, while blue represents technology, confidence, and professionalism.

3. **Fonts:**
   I used Arial, sans-serif for readability and a clean professional appearance.

4. **Personality and Field:**
   The design reflects my interest in technology by using a simple layout, strong contrast, and a modern color scheme.

5. **Homepage Layout:**
   The homepage includes a navigation bar, profile image, name, professional title, short bio, and resume download button.

6. **Project Sections:**
   Instead of a separate projects page, my GitHub link allows visitors to view my projects directly.

7. **Mobile-Friendly Design:**
   The website uses flexible layouts, grid structures, and responsive spacing to support different screen sizes.

8. **Visual Hierarchy:**
   Large headings, blue highlights, section titles, cards, and buttons guide visitors through the most important information.

9. **Consistency:**
   Consistency is maintained through the same navigation bar, color scheme, typography, spacing, and button styles across all pages.

10. **Accessibility:**
    I used readable font sizes, strong color contrast, clear navigation, descriptive image alt text, and simple page structure.

11. **Images and Icons:**
    I used personal images and school logos to make the website more personal and visually engaging.

12. **Design Inspiration:**
    My design was inspired by modern developer portfolio websites that use clean layouts, dark themes, clear navigation, and strong call-to-action buttons.

## Part 3: Interactivity

1. **Interactive Elements:**
   The website includes navigation links, hover effects, buttons, a resume download button, external GitHub/LinkedIn links, and a contact form.

2. **Contact Form:**
   The contact form uses FormSubmit to send messages directly to my email address. Users can enter their name, email, and message.

3. **JavaScript Features:**
   JavaScript is used to display a success message after a user submits the contact form.

4. **User Feedback:**
   Users receive feedback through hover effects and a success message after submitting the form.

5. **User Experience Improvement:**
   Interactivity improves the user experience by making the website easier to navigate, allowing users to contact me directly, and making actions feel clear and responsive.
